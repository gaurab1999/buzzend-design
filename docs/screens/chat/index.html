Interactions to try

Chat list: switch All / Friends / Challenges tabs · type in search · long-press (or right-click) a row → Mute / Mark read / Pin / Delete · ⋮ → Support (topic picker → support chat) / Mark all read.
Conversation: long-press a message → quick-react row + Reply / Copy / Forward / Edit / Delete · tap a reaction chip → who-reacted sheet · tap "Seen by" on a group message → seen-by list · ⋮ → mute / search / members / block / report / delete · + → attach photo/GIF · type @ in a group for mention autocomplete · tap an image → fullscreen.
